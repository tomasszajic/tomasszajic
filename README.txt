install instructions:
---------------------
1) download qt-opensource-windows-x86-5.14.0.exe from https://download.qt.io/archive/qt/5.14/5.14.0/
2) open QtCreator IDE
3) open monitor.pro and client.pro files (these are handled by QMake)
4) Kits selection: mingw32 compiler were chcecked. MSVC should work as well. Clicking on details: debug and release were checked.
5) Build and run.

OS used was Windows 7. Both programs were run on localhost only, using different ports. The table in the Client is configurable.
